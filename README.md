VOYAGE & SAVEURS

Ce projet a été réalisé dans le cadre du module HTML/CSS et consiste à créer un site web complet, multi-pages et entièrement codé sans JavaScript.
J’ai choisi de développer un site autour du thème “Voyage & Saveurs”, qui invite l’utilisateur à découvrir la cuisine de plusieurs pays à travers une expérience visuelle immersive.

L’objectif principal du site est de présenter, pour chaque pays, quelques plats représentatifs tout en travaillant différents aspects essentiels du développement web : structure HTML, organisation des fichiers, cohérence graphique, mise en page, et interactions simples basées uniquement sur le CSS.

Structure du site

Le site se compose de plusieurs pages :
- Accueil : introduction générale du projet et navigation vers les différents pays.
- France : présentation de quatre plats français avec un système de survol affichant une description.
- Italie : mise en avant des plats sous forme d’assiettes cliquables, avec une carte descriptive à droite.
- Japon, Maroc, Turquie : chaque page possède une ambiance visuelle unique, adaptée au pays, et un système d’interaction différent (hover, boutons radio, cartes dynamiques, etc.).
- Page de contact : formulaire simple permettant à l’utilisateur d’envoyer un message fictif.

Approche graphique

Chaque pays possède une identité visuelle spécifique :
- Un fond, une palette de couleurs et des typographies adaptées.
- Une mise en page différente pour éviter la répétition et montrer plusieurs techniques CSS.
- Des images optimisées et intégrées dans des layouts créatifs (assiettes circulaires, cartes animées, superposition d’images, etc.).

L’objectif était de créer une expérience agréable, cohérente et moderne, tout en respectant les contraintes du projet.

Interactions en CSS

Comme le JavaScript était interdit, j’ai utilisé des techniques avancées en CSS, notamment :
- Les inputs radio pour afficher dynamiquement les descriptions de plats.
- Les effets hover pour révéler du texte ou accentuer une image.
- Les transitions et transformations pour donner du mouvement aux éléments.
- Les ombres, dégradés et bordures pour rendre l’interface plus vivante.
- 
Ces méthodes montrent qu’il est possible de créer un site interactif uniquement avec du HTML et du CSS.

Organisation du code

Le projet est structuré de manière claire :
- Un fichier HTML par page.
- Un fichier CSS dédié pour chaque pays.
- Un dossier contenant toutes les images classées par pays.
- Une navigation identique sur toutes les pages pour améliorer l’expérience utilisateur.
- J’ai également travaillé sur la cohérence visuelle, l’espacement, l’harmonie des couleurs et la lisibilité du texte.

Ce que j’ai appris

Ce projet m’a permis de :
- Maîtriser la structuration d’un site multi-pages.
- Utiliser des mises en page complexes
- Créer des interactions sans JavaScript.
- Gérer l’alignement, les images, le responsive basique et la cohérence graphique.
- Améliorer ma manière d’organiser un projet web complet.

Comment utiliser le site

Il suffit d’ouvrir le fichier accueil.html dans un navigateur pour naviguer dans l’ensemble du site.
